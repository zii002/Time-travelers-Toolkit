This project is part of the Codecademy Python curriculum and focuses on combining multiple Python concepts into a single interactive script.
The goal is to simulate a time travel experience by working with dates, calculations, randomness, and modular code design.

-Objectives-
Create and use a custom Python module
Work with date and time data
Perform precise calculations
Implement randomness for dynamic outputs
Generate a personalized time travel message

-What I Learned-
Using Python modules (datetime, random, decimal)
Creating and importing custom modules
Handling real-time data (current date & time)
Writing reusable and structured code
Combining multiple concepts into one project
